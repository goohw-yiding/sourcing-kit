## 이번 버전 주요 변경사항

### 1. 하단 버튼 가림 현상 수정
- 소싱 상세 화면에서 주황색 "발주하기" 버튼이 하단 네비게이션 메뉴에 가려지는 문제 수정
- `pb-28` → `pb-40` 으로 하단 패딩 증가 (전체 뷰 3곳 공통 적용)

### 2. C/O 원산지증명 계산 개선
- **포함/미포함 토글**: "원가포함" / "포함하지않음" 텍스트로 상태 명확히 표시
- **C/O 총 비용 입력**: 발급 비용 총액 입력 (예: 50,000원)
- **총 주문수량 입력**: 이 CO를 몇 개 수량에 배분할지 별도 입력
  - 여러 상품과 함께 발주 시 → 전체 합산 수량 입력
  - 이 상품만 단독 발주 시 → 이 상품 수량만 입력
- **개당 자동계산**: C/O 총액 ÷ 총 주문수량 = 개당 CO 비용 실시간 미리보기

### 3. 중국어 회화 음성(TTS) 개선
- **기존 문제**: `window.speechSynthesis` — 모바일에 중국어 음성 팩 없으면 무음
- **1차 (Google TTS 프록시)**: `/api/tts` 서버 라우트로 Google TTS 음성 스트리밍
  - CORS 없이 자연스러운 중국어 발음 재생
  - 결과 24시간 캐시
- **2차 폴백**: Web Speech API, zh-CN 음성 명시적 선택
- **UI**: 재생 중 ⏸ 표시 / 다시 누르면 정지 / 오류 시 ⚠️ 표시

### 4. 소싱 현황 위젯 재설계 (홈 화면)
- **기존**: 검토중 / 제안완료 / 발주완료
- **변경**: 🔍 시장조사중 / 📦 발주진행중 / ✅ 입고완료
  - 시장조사중: sourcing + proposed 상태 합산
  - 발주진행중: ordered + shipping 상태 합산
  - 입고완료: arrived 상태
- 제안(proposal)은 소싱 파이프라인이 아닌 별도 기능임을 명확히 분리

### 5. 소싱 목록 필터 탭 정리
- **제거**: 📋 제안 탭 제거
- **변경**: 🔍 시장조사 탭이 sourcing + proposed 상태 모두 포함
- **최종 탭**: 전체 / 🔍시장조사 / 📦발주 / 🚢선적 / ✅입고

### 6. 더보기 — 바이어에게 제안 전용 섹션 신설
- 제안 기능을 더보기 메뉴 목록 한 줄에서 → **전용 카드 섹션**으로 격상
- **새 제안서 버튼**: 탭 누르면 `/proposals?new=1` → 바이어 선택 마법사 바로 시작
- **최근 제안서 3건 미리보기**: 목록 들어가지 않아도 👁 미리보기 / 📤 링크 복사 가능
- **바이어 관리** 링크도 같은 섹션에 묶어 맥락 통일
- 도구 섹션 재정리: 공급업체 / 발주이력 / HS코드 / 원가계산기

## 소싱 상품 상태 흐름

```
시장조사(sourcing) → 발주완료(ordered) → 중국내선적(shipping) → 입고완료(arrived)
```
- `proposed` 상태: 바이어에게 제안한 상품 (소싱 흐름과 독립)
- 제안서 기능: 더보기 → 바이어에게 제안
